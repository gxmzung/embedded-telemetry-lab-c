# Roadmap

## v0.1 Initial Telemetry Diagnostics Lab

- CSV telemetry packet parser
- voltage anomaly detection
- temperature anomaly detection
- signal strength anomaly detection
- session summary report

## v0.2 Protocol Validation

- CRC16 validation integration
- invalid packet counter
- malformed packet detection
- packet loss simulation

## v0.3 Field Operation Mode

- UART-style log parser
- binary packet inspector
- mission event timeline
- warning report export

## v1.0 Target

A small C-based embedded telemetry diagnostics toolkit that can parse packets, detect anomalies, validate protocol integrity, and summarize mission-session health.
